## 🚀 빠른 시작

```bash
# 1. 저장소 클론
git clone https://github.com/your-username/vibe-coding-bootcamp.git
cd vibe-coding-bootcamp

# 2. index.html 브라우저에서 열기
open index.html

# 또는 로컬 서버 실행 (Python)
python -m http.server 8080
# → http://localhost:8080 접속
```

### GitHub Pages 배포

1. 이 저장소를 Fork
2. Settings → Pages → Source: `main` branch, `/ (root)` 선택
3. `https://your-username.github.io/vibe-coding-bootcamp` 접속!
